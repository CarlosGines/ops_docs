Application Server (IIS)

-	Microsoft .NET Framework 4.5.2


Workplace Systems
----------------------------
-	Microsoft Internet Explorer 8 or higher
-	Mozilla FireFox 5 or higher (Windows SSO is not supported by browser)
-	Google Chrome
